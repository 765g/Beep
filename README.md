## ✨ Features

### 🎯 Combat
- **Silent Aim** - Configurable FOV, smoothness, and target part
- **Target Lock** - Press Q to lock onto targets
- **Triggerbot** - Auto-shoot when cursor is on enemy
- **Auto Shoot** - Automatic shooting for locked targets
- **Ultra Rapid Fire** - Extreme fire rate (50 shots/sec)
- **Kill Aura** - Auto-attack nearby enemies
- **Ragebot** - Full automatic targeting with game profiles (Rivals, Arsenal, Jailbreak)
- **Team Check** - Avoid friendly fire

### 👁️ Visuals
- **ESP** - Names, distance, user IDs
- **3D Chams** - Highlight players through walls
- **Box ESP 2D** - 2D boxes around players
- **Skeleton ESP** - Draw player skeletons
- **Tracers** - Lines from screen bottom to players
- **Health Bars** - Show player health
- **Head Dot** - Red dot on enemy heads

### ⚡ Physics
- **Speed Hack** - Move faster (toggle with key)
- **Jump Power** - Adjustable jump height
- **Fly** - Press E to fly (WASD + Space/Shift)
- **NoClip** - Walk through walls (F2 to toggle)
- **Click Teleport** - Ctrl+Click to teleport

### 🔧 Misc
- **Fullbright** - See in dark areas
- **Remove Fog** - Clear visibility
- **Infinite Jump** - Jump in mid-air
- **Anti-AFK** - Stay connected
- **FOV Changer** - Adjust field of view
- **Watermark** - Draggable status display
- **Theme Colors** - 8 accent color options
- **Teleport to Player** - Quick teleport by name

## ⌨️ Keybinds

| Key | Function |
|-----|----------|
| `INSERT` | Toggle menu |
| `Q` | Lock/unlock target |
| `E` | Toggle fly (when enabled) |
| `F2` | Toggle noclip (when enabled) |
| `Left Ctrl` | Speed boost / Click TP |
| `END` | Panic key (disable all) |

## 🎨 Theme

- Background: `RGB(12, 12, 16)`
