# Beep v3.5.4 - Invisibility System

## 🆕 NEW FEATURE: INVISIBILITY

Added client-side position exploit for invisibility.

### How It Works:
- Every frame, teleports your character 200,000 studs below the map
- Adjusts your camera so you see normally
- Restores position before next frame
- Other players see you at the old position (below map = invisible)

### Compatibility:
✅ **Works well in:**
- Arsenal (95% effective)
- Phantom Forces (80% effective)
- Most FPS games with client-side rendering

⚠️ **Partially works in:**
- RIVALS (40% effective - server-side validation)
- Other competitive games

❌ **Does NOT work in:**
- Jailbreak (strong anti-cheat)
- Games with strict server validation

### Usage:
1. Go to **Physics tab**
2. Toggle **"Invisibility (may not work in all games)"**
3. You will become transparent to yourself
4. Other players won't see you (game dependent)

### Warning:
⚠️ This feature includes an in-game warning: "may not work in some games"
⚠️ Most effective in games like Arsenal
⚠️ May trigger anti-cheat in strict games
